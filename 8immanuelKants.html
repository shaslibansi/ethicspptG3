-The concept of the "categorical imperative": Act only according to maxims that can be universalized.


The Categorical Imperative

Kant’s key principle:

“Act only according to that maxim through which you can at the same time will that it should become a universal law.”

  • Maxim: A personal rule of action (e.g., “I will lie to gain money”).
  • Universalization Test: Ask if everyone could act on the same rule without contradiction.
  • If everyone lied, promises would lose meaning—therefore lying is immoral.
  • Moral Law: Only actions that could logically and consistently be universalized are morally right.
